1

Join a Node to the Cluster

Register a new worker node with its capabilities:

curl -X POST http://localhost:8787/api/v1/nodes/join \
  -H "Content-Type: application/json" \
  -d '{"node_name": "worker-1", "capabilities": ["python", "data-processing"]}'

The node will appear in the dashboard within seconds.

2

Submit a Task

Create a new task for the cluster to process:

curl -X POST http://localhost:8787/api/v1/tasks \
  -H "Content-Type: application/json" \
  -d '{"title": "Process dataset", "requires": ["python"]}'

Tasks are automatically assigned to nodes with matching capabilities.

3

Monitor Progress

Watch tasks progress in real-time via the Dashboard or API:

curl http://localhost:8787/api/v1/status | jq
# Returns cluster summary, task states, and node health

The dashboard auto-refreshes every 5 seconds.

4

View Workflow Dependencies

Visualize task dependencies and execution order:

curl http://localhost:8787/api/v1/workflow/graph | jq
# Returns nodes and edges for the dependency graph

Use the Dashboard's Workflow Graph section for a visual representation.

API Reference

GET /api/v1/status Cluster status
POST /api/v1/nodes/join Register node
GET /api/v1/nodes List nodes
POST /api/v1/tasks Submit task
GET /api/v1/tasks List tasks
POST /api/v1/tasks/{id}/complete Complete task
POST /api/v1/tasks/{id}/fail Fail task
POST /api/v1/leases Create lease
DELETE /api/v1/leases/{id} Revoke lease
GET /api/v1/recovery/stats Recovery stats
GET /api/v1/workflow/graph Dependency graph
GET /api/v1/sync/status Sync status
